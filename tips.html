终端小技巧

一些让命令行操作更顺手的小技巧，持续记录中。

  • Ctrl + A 光标跳到行首
  • Ctrl + E 光标跳到行尾
  • Ctrl + W 删除光标前一个单词
  • Ctrl + U 删除光标前所有内容（到行首）
  • Ctrl + K 删除光标后所有内容（到行尾）
  • Ctrl + R 搜索历史命令（再按一次继续搜索）
  • Ctrl + L 清屏（等同于 clear 命令）
  • Ctrl + C 终止当前正在运行的命令
  • Ctrl + D 退出当前终端会话
  • Ctrl + _ 撤销上一次编辑操作
  • Tab 自动补全路径、文件名或命令
  • Esc . 复制上一条命令的最后一个参数
  • !! 重复执行上一条命令
  • !$ 引用上一条命令的最后一个参数（同 Esc .）
  • cd - 回到上一个目录
