# Static Pages

这是一个学习过程中随手搭起来的小项目，用来练习静态页面、基础 HTML/CSS/JavaScript，以及 GitHub Pages 的发布流程。

项目不追求复杂功能，主要是边学边试，把一些小想法做成能打开、能点击、能继续改的页面。

## 页面

- `index.html`：首页 Demo，包含简单按钮交互和当前时间显示。
- `tips.html`：记录一些常用终端快捷键和小技巧。
- `fun-facts.html`：整理几个和终端命令相关的冷知识。
- `shuangpin.html`：微软双拼键盘图，展示微软双拼方案的按键布局，支持现场试验输入。

## 本地查看

直接用浏览器打开 `index.html` 就可以查看页面。

也可以用一个本地静态服务查看，例如：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## 学习目标

- 熟悉静态页面的基本结构。
- 练习 CSS 布局和简单视觉样式。
- 写一点点 JavaScript 交互。
- 了解多页面之间的跳转。
- 练习 Git 和 GitHub Pages 的基本工作流。
